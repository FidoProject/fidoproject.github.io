Trains a neural network on a training set until the target error level is reached by modifying the weights and structure of the neural network.

Edits the weights and sturcture of the neural network until its error in predicting the correctOutput of each input reaches the value of targetErrorLevel or the number of training cycles reaches the value of maximumIterations. NOTE: If learning rate is not low enough, the weights of the neural network may got to infinity due to the nature of backpropagation.

Parameters
networkthe neural network to be trained
inputa vector of neural network inputs; each element in input, should have a corresponding output in correctOutput
correctOutputnetwork is trained to output an element of correctOutput when fed a corresponding element of the input vector
