The Fido Project

An open source C++ machine learning library targeted towards embedded electronics and robotics.

about fido

Fido is an open-source, highly modular C++ machine learning library for embedded electronics and robotics. Fido includes implementations of neural networks, reinforcement learning methods, and genetic algorithms. Fido is especially suited for robotic contexts, as it is written in C++ with minimal use of the standard library, comes packaged with a robotic simulator, and provides and easy interface in which to write robotic drivers. Fido also comes packaged with a human-trainable robot control system as described in Truell and Gruenstein.

getting started

First, download the latest release from Github. Navigate to the library's folder and build the library.

$ cd Fido && make install

And you're done! To use Fido, just include Fido.h in your C++ code. For more information please visit the docs.

quick example

Here is a quick example of using the Fido library to train a robot to perform line following.

rl::FidoControlSystem learner = rl::FidoControlSystem(1, 2, {-1, -1}, {1, 1});
Simlink simulator;
double maxMovement = 20;
int goodIterations = 0;

// Robot has to stay on the line for 5 reward iterations
while(goodIterations < 5) {
	// Choose an action. Give the side of the line that the robot is on as the state
	rl::Action action = learner.chooseBoltzmanAction({ simulator.isLeftOfLine() }, 0.2);

	// Execute the action. Note change in distance from line
	double originalDistance = simulator.distanceFromLine();
	simulator.robot.setPosition(simulator.robot.getPosition() + sf::Vector2f(action[0]*maxMovement, action[1]*maxMovement));
	double newLineDistance = simulator.distanceFromLine();

	// Give the movement towards the line as reward. Weight by max possible movement.
	learner.applyReinforcementToLastAction((fabs(originalDistance) - fabs(newLineDistance)) / sqrt(2*pow(maxMovement, 2)), { simulator.isLeftOfLine() });
	
	std::this_thread::sleep_for(std::chrono::milliseconds(10));

	// The robot has stayed on the line if it is within 80 pixels of it
	if(simulator.distanceFromLine() < 80) goodIterations++;
	else goodIterations = 0;
}

who is fido

Fido was created by programmers Joshua Gruenstein and Michael Truell from the Horace Mann School. The project was started as a universal robot control system using limited feedback, which can be read about here. Fido still continues to be developed in this direction.
